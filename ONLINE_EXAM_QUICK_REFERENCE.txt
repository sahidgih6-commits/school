┌────────────────────────────────────────────────────────────────┐
│ 🔌 API ENDPOINTS                                               │
└────────────────────────────────────────────────────────────────┘

  TEACHER ENDPOINTS:
  ──────────────────────────────────────────────────────────────
  POST   /api/online-exams              Create new exam
  GET    /api/online-exams              List all exams
  GET    /api/online-exams/{id}         Get exam details
  PUT    /api/online-exams/{id}         Update exam
  DELETE /api/online-exams/{id}         Delete exam
  
  POST   /api/online-exams/{id}/questions            Add question
  PUT    /api/online-exams/{id}/questions/{qid}      Update question
  DELETE /api/online-exams/{id}/questions/{qid}      Delete question
  
  GET    /api/online-exams/{id}/analytics            View analytics

  STUDENT ENDPOINTS:
  ──────────────────────────────────────────────────────────────
  GET    /api/online-exams                   Get published exams
  POST   /api/online-exams/{id}/start        Start exam
  POST   /api/online-exams/attempts/{id}/answer    Save answer
  POST   /api/online-exams/attempts/{id}/submit    Submit exam
  GET    /api/online-exams/attempts/{id}/results   View results
  GET    /api/online-exams/{id}/my-attempts        My attempts

  POST /api/online-exams/{exam_id}/questions

    "question_text": "What is Newton's First Law?",
    "option_a": "Force = mass × acceleration",
    "option_b": "Object at rest stays at rest",
    "option_c": "Action = Reaction",
    "option_d": "Energy is conserved",
    "correct_answer": "B",             // A, B, C, or D
    "explanation": "Optional explanation here..."

  Step 3: PUBLISH EXAM
  ──────────────────────────────────────────────────────────────
  PUT /api/online-exams/{exam_id}
  {
    "is_published": true
  }

┌────────────────────────────────────────────────────────────────┐

└────────────────────────────────────────────────────────────────┘

  Step 1: VIEW AVAILABLE EXAMS
  ──────────────────────────────────────────────────────────────
  GET /api/online-exams
  
  Step 2: START EXAM
  ──────────────────────────────────────────────────────────────
  POST /api/online-exams/{exam_id}/start
  → Returns: attempt_id, questions, timer starts
  
  Step 3: ANSWER QUESTIONS
  ──────────────────────────────────────────────────────────────
  POST /api/online-exams/attempts/{attempt_id}/answer
  {
    "question_id": 123,
    "selected_answer": "B"
  }
  
  Step 4: SUBMIT (before timer or auto-submit)
  ──────────────────────────────────────────────────────────────
  POST /api/online-exams/attempts/{attempt_id}/submit
  {
    "auto_submit": false
  }
  → When timer expires, auto_submit: true
  
  Step 5: VIEW RESULTS
  ──────────────────────────────────────────────────────────────
  GET /api/online-exams/attempts/{attempt_id}/results
  → Instant results with explanations

┌────────────────────────────────────────────────────────────────┐
│ ⏱️ AUTO-SUBMIT FEATURE                                         │
└────────────────────────────────────────────────────────────────┘

  HOW IT WORKS:
  ─────────────────────────────────────────────────────────────
  1. Timer starts when exam begins
  2. JavaScript countdown shows MM:SS
  3. When timer reaches 0:00 → auto-submit
  4. Server validates time hasn't been manipulated
  5. Exam marked as "auto_submitted: true"
  6. Results shown immediately

  RESUME PROTECTION:
  ─────────────────────────────────────────────────────────────
  • If student refreshes, checks elapsed time
  • If time expired → must submit
  • If time remaining → resumes with correct timer

┌────────────────────────────────────────────────────────────────┐
│ ⚙️ SYSTEM RULES                                                │
└────────────────────────────────────────────────────────────────┘

  EXAM LIMITS:
  ──────────────────────────────────────────────────────────────
  • Max 40 questions per exam
  • Each question MUST have 4 options (A, B, C, D)
  • One correct answer required (A, B, C, or D)
  • Explanation is optional
  • Each question = 1 mark (customizable)

  TIMING:
  ──────────────────────────────────────────────────────────────
  • Duration set in minutes
  • Timer accurate to 1 second
  • Auto-submit at exactly 0:00
  • No answers accepted after time expires

  SCORING:
  ──────────────────────────────────────────────────────────────
  • Correct answer = Full marks
  • Wrong answer = 0 marks
  • No answer = 0 marks
  • Percentage = (Score / Total) × 100
  • Pass/Fail based on pass_percentage
